使用提示

  • 选择币种后会自动从自有 API 获取对人民币（CNY）的汇率并填入“自定汇率”。
  • 你可以手动覆盖汇率或直接用自动汇率。
  • 生成的分享链接会把当前所有输入编码到 URL，打开即可恢复。
  • 交易日期：指交易生效的起算日（达成并生效/交割之日），用于计算到期前剩余天数；不同理解可能带来±1天差异，请以实际流程为准。
